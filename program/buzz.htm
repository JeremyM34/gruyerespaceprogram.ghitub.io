Buzz

Buzz is our test platform for the GNC of Colibri. It is an electric autonomous flying vehicle, which allows to perform a lot of back to back tests in a short time.

Design

This vehicle has been designed as a test platform. FDM 3D printing was extensively used for its manufacturing, which allows for a good repairability and modularity. Multiple configurations as well as various sensors can thus be analyzed to improve the GNC of Colibri.

Control algorithms

To improve our control algorithms, we are developing a simulation in parallel to the PCB software.
The simulation and code architecture have been designed in a manner that allows an easy transfer to the Colibri project.

Electronics

Following the development of multiple printed circuit boards for our TVC rockets, this PCB allows Buzz to be controlled in an autonomous way while sharing its flight data via WIFI. This PCB embeds an IMU, a barometer, a GPS, a LIDAR and an optical flow module.
